# wildberries Scraper

<b> Этот парсер достает информацию со страниц с товарами </b>


<h4>Извлекаемые данные:</h4>
<ul> 
  <li>Полное имя товара </li>
  <li>Бренд</li>
  <li>Текущая цена</li>
  <li>Обычная цена</li>
  <li>Ссылка на страницу с товаром</li>
  
</ul>
<hr> 
<b> Всю полученную информацию сохраняет в файл result.csv </b>
